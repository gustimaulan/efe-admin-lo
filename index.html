Pembatasan Admin (Config-based)

Pembatasan Reguler:

  • Admin 1: Tidak dapat memproses campaign 247002, 247003
  • Admin 2: Tidak dapat memproses campaign 247001
  • Admin 3: Tidak dapat memproses campaign 247004
  • Admin lainnya: Dapat memproses semua campaign (kecuali pembatasan khusus)

Pembatasan Kondisional (Admin 99):

Ketika Admin 99 dipilih:

  • Admin 1 & Admin 5: Hanya dapat memproses campaign 247001
  • Admin lainnya: Mengikuti pembatasan reguler mereka
  • Admin 99: Tidak dapat memproses campaign apapun

Catatan: Ketika Admin 99 aktif, Admin 1 dan Admin 5 akan dibatasi hanya untuk campaign 247001 saja.

  • Pembatasan dikelola secara otomatis melalui konfigurasi server
  • Sistem akan otomatis memfilter admin berdasarkan pembatasan campaign
  • Conditional restrictions memiliki prioritas lebih tinggi dari pembatasan reguler

Automation Logs

1. Pemilihan Admin

  • Klik tombol untuk menambah admin baru
  • Klik tombol untuk menghapus admin
  • Pilih admin dari dropdown menu yang tersedia
  • Minimal harus memilih 1 admin

2. Memulai Proses

  • Pastikan semua admin telah dipilih
  • Klik tombol "Submit" untuk memulai proses
  • Proses dapat dilihat di tab "Logs"
  • Form akan direset otomatis setelah proses selesai

3. Catatan Penting

  • Sistem akan otomatis memfilter admin berdasarkan pembatasan campaign
  • Proses dapat memakan waktu beberapa menit
  • Jangan tutup browser selama proses berjalan
  • Lihat tab "Logs" untuk detail proses dan admin yang dilewati
